Horror's Game
-
바이오 하자드4의 모작으로 텍스트 UI의 지시에 따라 게임을 진행하며 좀비들과 전투하고 아이템을 획득하는 게임입니다.

기능 구현
-

> ### Playable Character
>   - 걷기
>   - 달리기
>   - 조준
>   - 사격

> ### Zombie_AI
>   - 시야각
>   - 근접 공격
>   - 추격(이동)
>   - AI_FoundLocationSystem

> ### Level Object
>   - Collision Object

> ### UMG
>   - 인벤토리
>     - 아이템 획득 시 아이템 항목 생성
>     - 아이템 항목 클릭 시 아이템 사용

>   - 체력바
>     - 원형 체력바 (머티리얼 & 블루프린트로 제작)
>   - Aim UI
